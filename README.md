# 🌱 잘잇지 : 가족 관리 앱

**부모님을 챙기고, 가족과 더 가까워지는 연결 서비스**

> 자녀가 부모님의 건강과 일정을 돌보고, 퀴즈를 통해 가족을 더 깊이 이해할 수 있도록 돕는 가족 관리 앱입니다.  
> 15개의 질문을 3개의 챕터로 나누어 자연스럽게 서로를 이해하고, 공감의 대화를 만들어갑니다.

---

## ✨ 주요 기능

### 🏠 홈 화면
- **일일 질문**: 부모님과 가족에 대한 질문에 매일 답변
- **한 번에 질문 풀기**: 총 15개 질문을 챕터별로 연속 답변
- **이구동성 퀴즈**: 객관식 퀴즈로 가족과의 공통점 확인
- **카테고리 선택**: "부모님에 대한 질문" / "가족에 대한 질문" 선택 가능

### 📝 질문 기록
- **자동 저장**: 한 번에 질문을 완료하면 질문 탭에 자동 저장
- **유형별 구분 아이콘**
  - 🔮 일일 질문
  - 📝 한 번에 질문
  - 🎲 이구동성 퀴즈
- **챕터 정보 표시**: 각 질문이 어느 챕터에 속하는지 뱃지로 확인
- **카테고리 색상 구분**: 부모/가족 질문을 색상으로 구분

### 📅 가족 캘린더
- **월간 뷰**: 시각적으로 일정 확인
- **일정 추가**: 자주 쓰는 일정 템플릿 제공
- **가족 관련 일정 관리**
  - 병원 진료, 약 복용, 안부 연락, 기념일 등 관리

### 👨‍👩‍👧‍👦 가족 관리
- **가족 멤버 구분**: 부모/자녀 역할 설정
- **가족 갤러리**: 가족 사진 공유 및 저장
- **멤버별 답변 확인**: 가족 구성원이 어떤 질문에 어떻게 답했는지 확인 가능

---

## 🎮 사용 방법

1. **시작하기**: 메인 화면에서 ‘한 번에 질문 풀기’ 클릭
2. **역할 선택**: 부모님 또는 자녀 선택
3. **질문 답변**: 총 3개 챕터, 15개 질문에 차례대로 답변
4. **결과 확인**: 답변 완료 후, [질문] 탭에서 기록 확인

---

## 🧑‍💻 기술 스택

| 영역 | 기술 |
|------|------|
| Frontend | React + TypeScript + Vite |
| UI 프레임워크 | Tailwind CSS + Shadcn/ui |
| 애니메이션 | Framer Motion |
| 라우팅 | React Router |
| 상태관리 | React Hooks |
| DB/백엔드 | Supabase (예정) |
